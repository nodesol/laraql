Schema Basics

LaraQL uses **Code-First Discovery**. Instead of maintaining a separate .graphql file, your PHP classes become the source of truth. LaraQL scans these classes and generates the SDL for Lighthouse on the fly.

The #[Model] Attribute

When you tag an Eloquent model with #[Model], LaraQL creates an entire CRUD suite automatically.

Automated SDL Generation:

  • Full GraphQL Type.
  • Input type for data.
  • find & all Queries.
  • Full CRUD Mutations.

Types & Fields

Use #[Type] and #[Field] for granular control over what is exposed. 

use LaraQL\Attributes\{Type, Field};

#[Type]
class UserProfile
{
    #[Field]
    public string $bio;

    #[Field]
    public function avatarUrl(): string 
    {
        return "https://cdn.example.com/" . $this->id;
    }
}
